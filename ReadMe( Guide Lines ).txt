CRYPTOCURRENCY : A cryptocurrency, crypto-currency, or crypto is a digital currency designed to work as a medium of exchange througha computer network that is not reliant on any central
 authority, such as a government or bank, to uphold or maintain it.

STEPS TO RUN THIS CRYPTO APP :

1] Extract the Zip file using proper unzip software.

2] Open the APP folder and open command prompt ther ( Note : the path/adress in command prompt should be same as of the APP folder path/address),

3] Run command ( npm i ) on terminal. Wait for the installation to get complete.

4] Run command ( npm start ) on terminal and wait for the application to open in ur default browser
